# 🌸 Pink Calculator

## Deskripsi

Proyek ini merupakan website kalkulator sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript. Kalkulator ini dapat digunakan untuk melakukan perhitungan dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian. Selain itu, website ini memiliki tampilan bertema pink dengan desain yang sederhana, modern, dan responsif.

## Fitur

- Operasi penjumlahan
- Operasi pengurangan
- Operasi perkalian
- Operasi pembagian
- Tombol AC (Clear)
- Tombol hapus (Backspace)
- Perhitungan desimal
- Tampilan responsif
- Animasi pada tombol

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript

## Struktur Folder

```
📁 kalkulator project

└── README.md
```

## Cara Menjalankan

1. Download atau clone repository ini.
2. Buka folder proyek.
3. Jalankan file `index.html` menggunakan browser.

## Tujuan

Proyek ini dibuat sebagai latihan dalam mempelajari dasar pengembangan website menggunakan HTML, CSS, dan JavaScript, serta untuk melatih pembuatan tampilan website yang responsif dan interaktif.
